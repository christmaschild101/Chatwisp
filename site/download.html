Windows Client

The Chatwisp Windows client is a native desktop application built with wxPython. It provides full keyboard and screen-reader accessibility.

System Requirements

  • Windows 10 or later (64-bit)
  • Internet connection
  • No additional software required — the executable is self-contained.

Download

Download Chatwisp.exe

Version 3.0.0 — Portable executable, no installation required

How to Use

  1. Download the executable using the link above.
  2. Double-click Chatwisp.exe to launch the application.
  3. Create an account or log in with an existing one.
  4. Browse forums, read and write posts in real time.
  5. Topic links open in your browser — copy the link from the app and paste it into a browser address bar.
